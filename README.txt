
AHM Branding Co. - HTML Website

How to Publish (Option 1: GitHub Pages)
---------------------------------------
1. Go to https://github.com and login
2. Create a new repo: 'ahmbrandingco'
3. Upload all these files
4. Go to Settings > Pages > Select main branch
5. Visit: https://yourusername.github.io/ahmbrandingco/

How to Publish (Option 2: Netlify)
---------------------------------------
1. Go to https://netlify.com and sign up
2. Click 'Add new site' > 'Deploy manually'
3. Drag & drop all files and publish
4. Site will be live at something like ahmbrandingco.netlify.app
